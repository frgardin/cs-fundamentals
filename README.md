# cs-fundamentals

Minha jornada de formação completa em Ciência da Computação, seguindo o currículo da [Universidade Brasileira Livre](https://github.com/Universidade-Livre/ciencia-da-computacao).

## Sobre

Sou desenvolvedor Java sênior sem formação formal em CC. Este repositório documenta meu estudo sistemático de todos os fundamentos teóricos — do zero até o equivalente a um bacharelado completo.

O objetivo não é colecionar certificados. É entender **por que** as coisas funcionam.

## Estrutura

```
cs-fundamentals/
├── curriculum.md           # Mapa completo do currículo com links
├── progress.md             # Progresso global
├── etapa-1/
│   ├── progress.md
│   └── <disciplina>/
│       ├── summaries/      # Resumos por tópico
│       ├── tests/          # Testes de conhecimento com gabarito
│       └── notes/          # Anotações livres
├── reviews/
│   ├── weekly/             # Revisões semanais
│   ├── per-etapa/          # Revisão ao concluir cada etapa
│   └── overall/            # Visão geral do progresso
└── resources/
    └── key-references.md   # Livros e referências complementares
```

Diretórios de etapas futuras são criados sob demanda, conforme o progresso avança.

## Metodologia

- **Resumos** em português (BR) com conexões explícitas a Java e backend
- **Testes** com distribuição 20% fácil / 60% médio / 20% difícil
- **Revisões** semanais e por etapa para identificar lacunas
- **Rastreamento de score** por disciplina — tópicos abaixo de 70% entram em revisão
